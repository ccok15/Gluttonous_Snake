# 玩法规则 V2

状态：已同步代码

## 1. 移动与输入

- 初始移动间隔：`90 ms/步`
- `移动加速` 每级按累计 `-15 ms` 计算
- `移动加速` 的抽取权重与其他普通升级保持一致

## 1.5 穿越术补充规则

- `穿越术` 生效时，撞墙会消耗 `1` 次机会并穿越到地图另一侧
- `穿越术` 生效时，撞到自身也会消耗 `1` 次机会并直接穿过，不会立刻判负
- 只要仍有穿越机会，蛇头会使用特殊颜色提示能力仍在生效
- 暴击豆不能被 `微型磁吸` 自动吸走，只能由蛇头主动吃到
- 局内新增暂停键：`P`
- 吃到豆子后，需要在豆子位置显示 `+分数` 飘字

## 2. 阶段、地图与时间

- 第 1 阶段初始剩余时间：`60 秒`
- 每次阶段完成后，不重置剩余时间，只在当前剩余时间基础上 `+60 秒`
- 地图尺寸每两个阶段扩一次，最大到 `60x60`
- 当前确认的阶段尺寸序列：
  - `20x20`
  - `20x20`
  - `28x28`
  - `28x28`
  - `36x36`
  - `36x36`
  - `44x44`
  - `44x44`
  - `52x52`
  - `52x52`
  - `60x60`
  - `60x60`
- 地图达到 `60x60` 后不再继续扩张

## 3. 普通豆规则

- 第 1 阶段同屏普通豆数量：`4`
- 之后每个阶段普通豆数量 `+4`
- 当前确认的普通豆数量序列：
  - `4`
  - `8`
  - `12`
  - `16`
  - `20`
  - `24`
  - `28`
  - `32`
  - `36`
  - `40`
  - `44`
  - `48`
- 暴击豆是额外特殊豆，不占普通豆名额

## 4. 暴击豆规则

- 暴击豆是额外特殊豆，场上最多同时存在 `1` 颗
- 暴击豆用明显不同的颜色和出现方式与普通豆区分
- 暴击豆基础刷新间隔为：`8 秒 + 0~4 秒随机`
- 暴击豆初始存在时间：`5 秒`
- 暴击豆会闪烁，并随着剩余时间变少而加快闪烁，最后消失
- 暴击豆得分采用左偏钟形分布：
  - 低分最常见
  - 中段分数偶尔出现
  - 高分稀有
- 暴击豆初始得分区间：`5 ~ 10`

## 5. 暴击豆升级分支

- `暴击豆下限`
  - 初始最低分为 `5`
  - 每次升级后最低分 `+3`
  - 不封顶
- `暴击豆上限`
  - 初始最高分为 `10`
  - 每次升级后最高分 `+10`
- `暴击豆留存`
  - 初始存在时间为 `5 秒`
  - 每次升级后存在时间 `+3 秒`

## 6. 当前需要重点试玩的风险

- 以当前 `90 ms/步`、`最低 30 ms/步`、`移动加速每级 -15 ms` 的组合来看，`移动加速` 在第 `4` 级就会触及速度下限，后续等级价值会明显下降。
- 普通豆数量改成 `4, 8, 12 ... 48` 之后，当前阶段目标分曲线可能会偏低，试玩后大概率还要继续调目标分。
- 每次通关都在剩余时间上 `+60 秒`，如果前几阶段过于容易，后面可能会出现时间滚雪球。
